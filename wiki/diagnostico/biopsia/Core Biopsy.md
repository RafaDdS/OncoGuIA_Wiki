# Core Biopsy

A Core Biopsy, também conhecida como biópsia por agulha grossa (BAG), é um procedimento ambulatorial utilizado para extrair pequenos cilindros de tecido de uma área suspeita, permitindo a análise arquitetural completa da lesão. O procedimento é obrigatoriamente guiado por métodos de [[diagnostico/imagem/index|imagem]], como ultrassonografia ou estereotaxia, garantindo precisão na coleta.

## Aplicações Clínicas e Tratamento
É o exame padrão-ouro para a confirmação diagnóstica inicial do câncer de mama. A principal utilidade da Core Biopsy é fornecer tecido suficiente (histologia) não apenas para confirmar a malignidade, mas para definir o tipo de tumor, o grau histológico e, crucialmente, avaliar os [[biomarcadores/index|biomarcadores]] preditivos (Receptores Hormonais, [[HER2]] e [[Ki67]]) que ditarão toda a estratégia de [[Tratamento Neoadjuvante (Conceito)|tratamento neoadjuvante]] ou cirúrgico.
